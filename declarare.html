 -->Un tablou este un grup de elemente de același tip care formează o mulțime ordonată.La fiecare element se poate face referire prin intermediul unui indice.Tipul de bază al tabloului este tipul comun tuturor elementelor sale.Un tablou arre asociat un nume.
-->Declararea unei variabile de tip tablou:

tip_dată nume[nr_elemente];


*"tip_dată nume"=reprezintă cuvântul cheie pentru un tip predefinit
*"[nr_elemente]"=reprezintă limita superioară a indicelui


-->La elementele unui tablou este permis accesul direct prin intermediul unui index, se specifică numele tabloului și între paranteze pătrate indicele.
Exemple:
int v[10];
int x[5];
float c[10];
-->Cu elementele tabloului se pot face toate operațiile admise de tipul de bază al tabloului.NU SUNT PERMISE atribuiri de forma "b" ia valoarea lui "a"(b=a)unde a și b tablouri.În acest caz se poate realiza atribuirea , dar trebuie facută pe componente(b[i]=a[i]).
